Harvest & Mapping Data

Harvest Data Instructions

Upload harvest data Excel file with sections (Open Field, Polyhouse C, etc.) and product quantities.

  • Format: Section headers with product rows showing harvest quantities
  • Example: "Open Field" section with "Iceberg Lettuce: 1682.30 kg"

Purpose: Harvest data is used to compare with sales data. If sales show 21434 kg but harvest shows only 1682.30 kg, the system will split: 1682.30 kg (inhouse) + 19751.70 kg (outsourced).

Mapping Data Instructions

Upload an Excel file with two columns:

  • Section: Section name (e.g., "Open Field", "Polyhouse C")
  • Product: Product name (e.g., "Beetroot Leaves", "Arugula(Rocket Lettuce)")

Purpose: This mapping is used to allocate VARIABLE COST subcategories (like "OPEN FIELD", "LETTUCE") to the correct products based on their harvest section.

Excel Upload Instructions

Upload an Excel file with the following columns:

  • Month: Format like "24-Apr" or "2024-04"
  • Particulars: Product name (e.g., "Arugula (Rocket Lettuce)")
  • Type: "In-house" or "Outsourced"
  • Inward Quantity: Quantity received/purchased (in kg)
  • Inward Eff. Rate: Rate per kg for inward quantity
  • Inward Value: Total value of inward quantity
  • Outward Quantity: Quantity sold (in kg)
  • Outward Eff. Rate: Selling rate per kg
  • Outward Value: Total sales value

Auto-Detection: The system will automatically detect inhouse production (when outward > inward).

P&L Upload Instructions

Upload a Purple Patch P&L Excel file with the following format:

  • Column A: Particulars (e.g., "Cultivation Expenses", "Office Expenses")
  • Column B: Amounts (e.g., "211658.45", "50000")
  • Period: Should contain date range like "1-Apr-24 to 30-Apr-24"

Allocation Logic:

  • I Items: 100% allocated to Inhouse products
  • O Items: 100% allocated to Outsourced products
  • B Items: Split by calculated ratio (default: 18.22% Inhouse, 81.78% Outsourced)
